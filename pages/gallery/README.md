# Assignment Gallery - Interactive Assignments

## 📋 Overview

This folder contains interactive TypeScript assignments where students can see their code in action through visual web pages.

## 🎯 How It Works

### For Students:

1. **Edit the Assignment File**: Open `Assignment1.ts` in VS Code
2. **Complete the TODOs**: Follow the instructions in the comments
3. **Build Your Code**: Run `npm run build` in the terminal (from the project root)
4. **View Your Work**:
   - Go to the Assignment View page to see your code
   - Click "See Your Code in Action!" to see the interactive demo

### For Teachers:

Each assignment has three components:

- **AssignmentX.ts** - The TypeScript file students edit
- **AssignmentX.js** - The compiled JavaScript (auto-generated by build)
- **AssignmentX.html** - The interactive visual page that imports the JS

## 🔨 Build Process

The build process is configured in `package.json`:

```bash
npm run build-assignments  # Compiles only assignment files
npm run build              # Full build including assignments
```

This compiles all `.ts` files in `pages/gallery/` to `.js` files in the same directory.

## 📝 Assignment 1: Restaurant Menu

Students create:

- Variables for restaurant info (name, tables, open status)
- Functions for greeting customers and calculating totals
- An interface for menu items
- An array of menu items

The `Assignment1.html` page displays:

- Restaurant header with name and status
- Interactive greeting form
- Visual menu grid
- Order calculator using student's functions

## 🚀 Deployment

The compiled `.js` files should be committed to Git so they're available on GitHub Pages after deployment.

## 💡 Tips

- Students must run `npm run build` after making changes to see updates
- The HTML page will show helpful error messages if code isn't compiled
- All student code uses ES2020 modules with `export` statements
