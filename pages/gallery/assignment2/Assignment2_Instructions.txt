🤴 ROYALS AND ROBBERS 🥷
================================

📝 Your Task:
Implement a series of interfaces and classes to fill TypeLand Castle with the royal family
and a band of robbers trying to overtake the land.

This assignment focuses on:
- Creating interfaces to define object shapes
- Extending interfaces to create hierarchies
- Implementing optional and readonly properties
- Creating objects that follow interface contracts
- Adding methods to interfaces

✅ What You Need to Do:

PART 1: BASE INTERFACES (Create the foundation)
------------------------------------------------
TODO 1: Create a base "Person" interface with these properties:
   - name: string
   - age: number
   - introduce: method that returns a string

TODO 2: Create a "Royal" interface that EXTENDS Person and adds:
   - title: string (e.g., "King", "Queen", "Prince")
   - readonly kingdom: string (cannot be changed after creation)
   - wealth: number
   - decree: method that takes a message string and returns a string

TODO 3: Create a "Criminal" interface that EXTENDS Person and adds:
   - alias: string (their criminal nickname)
   - crimesCommitted: number
   - isWanted: boolean
   - steal: method that takes a target string and returns a string

🎯 Expected Output:

💡 Tips:
- Make sure your introduce() methods return creative, character-specific messages
- The decree() method should return the royal's proclamation
- The steal() method should return a theft description
- Use proper TypeScript types for all parameters and return values
- Remember that "readonly" properties cannot be reassigned after creation
